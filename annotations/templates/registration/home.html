Welcome to Vogon Web

VogonWeb is a web application that allows scholars to annotate texts with contextualized relationship triples, so-called “quadruples.” Quadruples form semantic networks that, when uploaded to a repository, can be used to for example develop advanced search engines. Furthermore, various software tools can be developed to analyze and visualize these networks.
